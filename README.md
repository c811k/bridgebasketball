# Bridge Basketball

Bridge Basketball is a website created for players in Bridge Basketball League to check their individual stats, scores, schedule, team standing, and watch their games.

## Deployed Website

https://basketball.nlvcbridge.com

## Built With

* Express
* React.js
* Node.js
